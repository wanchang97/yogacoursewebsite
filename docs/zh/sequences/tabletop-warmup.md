# 四角板凳支撑稳定序列 (Tabletop Support Flow)

**目标**：建立手掌与大臂的支撑觉知，激活核心与背部，为下犬式和站立体式做准备。
**耗时**：约 10 分钟

## 序列体式 (Asanas)

1. **宽膝婴儿式 (Wide-Knee Child's Pose)**
   * 建立从手掌到臀部的连接。
   * *口令提示*：手臂向前延伸，感受背部的拉长。

2. **四角板凳 (Table Top / Box Neutral)**
   * 寻找手掌推地的根基。感受到身体的器官自然地被支撑住。
   * *口令提示*：手掌的虎口、五个手指的指根与地面接触。大臂微微外旋，激活大臂内侧。

3. **板凳猫牛式 (Tabletop Cat-Cow)**
   * 配合呼吸，流动脊柱。

4. **板凳扭转 (Tabletop Twist)**
   * 单手支撑，打开胸腔。

5. **猫式平衡 / 单腿单臂伸展 (Cat Balance / Bird-Dog)**
   * 抬起对侧手脚，感受身体其他部位的稳定和伸展。

6. **虎式 (Tiger Pose)**
   * 手向后抓脚踝，挑战平衡并打开前侧。

7. **下犬式准备 / 膝盖悬空板凳 (Hovering Tabletop)**
   * 强烈激活核心，体会手掌承重。

---

## 教学笔记 (Teacher's Notes)
* 本序列的核心是**根基建立**。
* 在这里要详细讲解手掌的支撑发力点（虎口压实），避免学生以后在下犬式里手腕疼。
* 提示：力量是不费力地从肩胛骨传导到手掌再到地面。
