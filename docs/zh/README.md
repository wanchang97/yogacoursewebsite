## 🎯 自我介绍
我是那由他无常，
“那由他”源自佛经，意为不可计量的无穷。庄子养生主里有“吾生也有涯，而知也无涯”的描述。我们内心有着对无穷无限的向往和追求，但是我们每个人每时每刻都是被限制的存在，我们希望修习瑜伽获得不止是身心的结合，更是关注自己的生命和外界的宇宙的连接。
"无常"是只生命都是成住坏空，没有恒常。面对无常的生命，我有更多的渴望能够主动寻求改变，也希望有更多智慧顺应世事的变化。借此提醒自己专注当下，能看清此刻内心的需求，能认识真正的自己，用有限的生命最大限度地完成自己。

我从疫情开始从keep软件视频跟练瑜伽，到后来积极的打卡练习瑜伽的高难度动作，再到现在希望追寻瑜伽的精神哲学。瑜伽帮我了强身健体，也渐渐助我磨练心性。

我将总结自己修习瑜伽的经验和感受，结合 [Donna Farhi][donna] 的器官支撑理论和 [YogaLili][lili] 的身心原则（呼吸、根基、中心），帮助大家更安全、更深入地练习瑜伽。我们可以向往高阶体式（比如肩倒立，头倒立，手倒立，轮式等），但不以完成高阶体式为评判标准，而是清晰展示通往高阶体式的中间过程，以及每个过程中的感受。

器官支撑理论：
在进行呼吸和每个人体式时，都可以从身体中心，从身体的器官开始感受。
1. 支撑时：将气体吸入到身体腹部的深处，带到身体内部的每个器官，感受到身体内部的空间。呼气时将气体从腹部排空，感受心脏，肺部，胃，小肠大肠等稳定的支撑住。感受到自己消化道的支撑。瑜伽的力量和健身的力量不同的是，瑜伽的力量是基于伸展，基于空间的力量，是一种全身协调的有内部器官发起的力量，而不是由外在的肌肉主导的力量。
2. 伸展时：从身体中心胸腔腹腔开始向上下或者左右主动伸展，主动拉长。不要纸关注到四肢的伸展。
3. 站立时：感受身体器官的平衡和稳定，像是一碗水，不过与向前（腰向前塌陷翘臀）或者向后倾斜驮背。
4. 扭转时：也是从器官身体中心开始的扭转。支撑时，先从器官开始觉察支撑，再扩展到背部，扩展到四肢。

呼吸：
呼吸是每个人时时刻刻都会做的事情。我们可以从观察数自己的呼吸开始了解自己自然的呼吸是什么样子。
呼吸连接意识和身体的练习：
吸气，感受身体向上，向前伸展，感受腹部背部前后左右同时的伸展。
呼气，感受身体向下的扎根，在放松，排空腹部深层的气体时身体的稳定，感受到手或者脚和地面的稳定的支撑。感受到大腿或者胳膊内侧外侧前侧后侧肌肉的稳定。


## 📚 夏季学期的课程结构
**那由他瑜伽 Nayota Yoga 10周大学体育课程**

前面部分是基础体式的练习。我们每节课讲会着重体会一个基础体式，并通过线下的串联感受在练习这些体式的过程中，我们的意识应该关注在哪些地方。通过刻意的练习变成一种习惯，最好变成顺其自然的反应。

1. 该体式究竟**是什么**---一张标准图示。
   
2. 该体式**背后思想**---因为标准是一个外在的平均意向，而我们的身体在严格意义上都是非标准件，我们应该如何将体式为己所用，转化成适合自己的体式。那什么是适合自己的体式？一个判断的方法，是能否在这个体式保持顺畅均匀的呼吸。如果可以，那么可以在自己稳定的状态下进一步尝试更多的幅度。如果呼吸开始闭塞，开始憋气去完成一个动作，那此时这个动作就是对你的身体而言还没有完全准备好的动作。瑜伽的思想不是努力，不是结果，而是感受，是过程。我们头脑里首先可以有一个最终的目标形状，但是从自己目前的身体状态到这个目标体式的过程是瑜伽真正在意的。
   
3. **辅助工具 (Practice Tools)**：我们可以为了做到这个体式采用的练习方法。
