## 📚 夏季学期的课程结构
**那由他瑜伽 Nayota Yoga 10周大学体育课程**

前面部分是基础体式的练习。我们每节课讲会着重体会一个基础体式，并通过线下的串联感受在练习这些体式的过程中，我们的意识应该关注在哪些地方。通过刻意的练习变成一种习惯，最好变成顺其自然的反应。

1. 该体式究竟**是什么样子**：一张标准图示。
   
2. 该体式**背后思想**：因为标准是一个外在的平均值，而我们的身体在严格意义上都是非标准件，我们应该如何将体式为己所用，转化成适合自己的体式。那什么是适合自己的体式？一个判断的方法，是能否在这个体式保持顺畅均匀的呼吸。如果可以，那么可以在自己稳定的状态下进一步尝试更多的幅度。如果呼吸开始闭塞，开始憋气去完成一个动作，那此时这个动作就是对你的身体而言还没有完全准备好的动作。瑜伽的思想不是努力，不是结果，而是感受，是过程。我们头脑里首先可以有一个最终的目标形状，但是从自己目前的身体状态到这个目标体式的过程是瑜伽真正在意的。
   
3. **实践序列**：可以感受这种背后思想获得更多空间的实践序列。
