# 那由他瑜伽 Nayota Yoga 

欢迎来到 **那由他瑜伽 Nayota Yoga** 数字化资源空间。

## 🎯 核心宗旨
“那由他”源自佛经，意为不可计量的无穷。庄子养生主里有“吾生也有涯，而知也无涯”的描述。我们内心有着对无穷无限的向往和追求，但是我们每个人每时每刻都是被限制的存在，我们希望修习瑜伽获得不止是身心的结合，更是关注自己的生命和外界的宇宙的连接。我的网名是那由他无常，是借此提醒自己宇宙无穷和生命的无常。希望自己能专注当下，能看清此刻内心的需求，能认识真正的自己，用有限的生命最大限度地完成自己。

我将总结自己修习瑜伽的经验和感受，结合 [Donna Farhi][donna] 的器官支撑理论和 [YogaLili][lili] 的身心原则（呼吸、根基、中心），帮助大家更安全、更深入地练习瑜伽。我们可以向往高阶体式（比如肩倒立，头倒立，手倒立，轮式等），但不以完成高阶体式为评判标准，而是清晰展示通往高阶体式的中间过程，以及每个过程中的感受。

## 📚 课程结构
**那由他瑜伽 Nayota Vinyasa Yoga Flow 10周进阶课程**
每一堂课在网页上由以下 4 个固定模块组成：
1. **逻辑图示 (Logic & Anatomy)**：可视化解剖逻辑。
2. **精华视频 (10-min Deep Dive)**：针对“空间感”或“支撑点”的10分钟重点讲解。
3. **理论基石 (Theoretical Foundation)**：Donna Farhi 身体原则与解剖学要点。
4. **实践工具 (Practice Tools)**：关键动作序列与推荐练习歌单。
