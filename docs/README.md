# 那由他瑜伽10周进阶课程

欢迎进入那由他瑜伽 Yoga Nayota 10周进阶课程数字化资源中心！

本课程旨在通过结构化的教学方案，将 [Donna Farhi](https://www.masterclass.com/classes/donna-farhi-teaches-yoga-foundations) 的器官支撑理论和 [YogaLili](https://www.youtube.com/@yogalili6775) 的身心原则（呼吸、根基、中心）结合起来，帮助大家更安全、更深入地练习瑜伽。本网站的体式配图（Asanas Figures）来源于 [PocketYoga](https://www.pocketyoga.com/)。我们可以向往高阶体式（比如肩倒立，头倒立，手倒立，轮式等），但不以完成高阶体式为评判标准，而是清晰展示通往高阶体式的中间过程，以及每个过程中的感受。

## 本地预览（写内容时推荐）

本站用 [Docsify](https://docsify.js.org/) 渲染 `docs/` 里的 Markdown。最简单的方法是使用系统自带的 Python 启动简易服务器。

在项目的最外层根目录（`YogaCourseWebsite/docs/`）打开终端，执行以下同一套标准指令即可：

```bash
python3 -m http.server 3000
```

启动后，在浏览器打开：👉 **[http://localhost:3000/](http://localhost:3000/)** 。

编辑并保存 `docs/zh/` 等各语言文件夹下的 `.md` 文件后，直接刷新浏览器页面即可看到最新效果。
