# 那由他瑜伽10周进阶课程

欢迎进入那由他瑜伽 Yoga Nayota 10周进阶课程数字化资源中心！

本课程旨在通过结构化的教学方案，将 [Donna Farhi](https://www.masterclass.com/classes/donna-farhi-teaches-yoga-foundations) 的器官支撑理论和 [YogaLili](https://www.youtube.com/@yogalili6775) 的身心原则（呼吸、根基、中心）结合起来，帮助大家更安全、更深入地练习瑜伽。我们可以向往高阶体式（比如肩倒立，头倒立，手倒立，轮式等），但不以完成高阶体式为评判标准，而是清晰展示通往高阶体式的中间过程，以及每个过程中的感受。
